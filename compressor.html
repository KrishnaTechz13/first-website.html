Frequently Asked Questions

Are my images uploaded to a server?

No, all processing happens directly in your browser. Your images never leave your device, ensuring complete privacy.

What compression quality should I use?

For web use, 70-80% quality provides the best balance between file size and visual quality. For print, use 90-95%.

What image formats are supported?

We support all common image formats including JPG, PNG, WebP, and BMP for input. Output formats include JPEG, PNG, and WebP.

Is there a file size limit?

There's no hard limit, but very large files (over 20MB) may cause performance issues on some devices.
